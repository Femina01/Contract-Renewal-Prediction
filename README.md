Contract Renewal Prediction – Quick Notes

Problem Statement

You are solving:
* Predict whether a customer will renew a contract
* And later use it to decide discount strategy


## Steps in Pipeline

1. Load Data
2. Data Cleaning
3. Feature Engineering
4. Feature Selection
5. Train-Test Split
6. Model Selection
7. Cross Validation
8. Model Training
9. Prediction
10. Evaluation
11. Save Model

## Learnings

* Don’t trust 100% accuracy (overfitting)
* Always check recall for business problems
* Feature engineering improves model
* Probability is more useful than prediction

---

## Final Output

* Predict renewal
* Suggest discount based on probability
